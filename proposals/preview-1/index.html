Recommended Path Forward

Given the runway constraints, the cleanest path is to start with the Distribution Test in May and decide month-by-month from there. Here are the two ways the next three months can play out:

Path 1 · Stay light

Three identical one-off tests

Run the $5,000 Distribution Test in May, then again in June (launch) and July (post-launch). No commitment between months — Hound13 can stop after May if results don't justify continuing.

Max total: $15,000 across 3 months
Path 2 · Scale up after May

Test in May, retainer for June + July

Run the $5,000 Distribution Test in May. If results are promising, switch to the full retainer for June + July to maximize the launch window with hands-on outreach and embargo coordination.

Total: $5,000 + $13,000–$14,000 = $18,000–$19,000
